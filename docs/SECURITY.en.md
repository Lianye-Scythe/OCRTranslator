# Security Policy

[繁體中文](../SECURITY.md)｜[简体中文](SECURITY.zh-CN.md)｜English

If you discover a security issue in OCRTranslator, please do not post exploitable details, secrets, or sensitive request content directly in a public issue.

## Reporting principles

Please include at least:

- a description of the issue
- affected scope
- reproduction steps
- whether login, API keys, or specific configuration is required
- a reviewed crash log if one exists and does not expose sensitive data

> Crash logs currently redact some local paths and common sensitive parameters, but you should still manually review them before sharing.
