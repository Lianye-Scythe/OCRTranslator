# PySide6 現代化 UI 方向

## 目標
把 OCR 翻譯器做成更像現代桌面產品，而不是傳統表單工具。

優先順序：
1. 主視窗第一眼質感提升
2. 快速操作更明確
3. 設定區塊從密集表單改為有層次的卡片
4. 翻譯浮窗更像閱讀面板

## 視覺方向
- 深色主題，偏 `Google AI Studio x Win11 utility`
- 大圓角卡片
- 低對比背景 + 高對比重點按鈕
- 淺藍/青藍作為主色，綠色作為成功行為
- 陰影用於卡片分層，不做過重玻璃擬真

## 版面策略
### 主視窗
- 頂部 hero card：產品名稱、說明、兩個最常用操作
- 下方分頁保留，但提高 tab 質感
- 設定頁改為雙欄：
  - 左：設定檔 / API
  - 右：閱讀設定 / 快速操作
- 監看頁改為大圖預覽 + 右側 log panel

### 翻譯浮窗
- 保持無邊框、置頂
- 加入更柔和陰影
- Header 與 body 區分更清楚
- 後續可追加：字距、行距、pin 按鈕、半透明模式

## 互動策略
- 快速操作按鈕放 hero card 與設定區卡片雙入口
- 狀態列保留於底部，作為單一回饋區
- `Ctrl + 滾輪` 繼續保留在浮窗內縮放字體

## 本輪實作範圍

- 雙欄設定頁
- 卡片陰影與新版色票
- 改善按鈕 hover / hierarchy
